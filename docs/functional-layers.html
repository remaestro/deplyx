🔄 Relations d'influence entre couches
🏗️
Physique
Câbles, ports, datacenters
→ Héberge → Connecte
🌐
Réseau
Switches, routeurs, VLANs
→ Achemine → Transporte → Segmente
🛡️
Sécurité
Firewalls, règles, VPN
→ Filtre → Protège → Autorise/Refuse
📦
Application
Services, bases de données
→ Dépend de → Consomme → Expose
📊
Observabilité
Monitoring, logs, métriques
→ Scrute → Visualise → Alerte
🏗️ Physique → 🌐 Réseau → 🛡️ Sécurité → 📦 Application → 📊 Observabilité

Une défaillance dans une couche impacte toutes les couches supérieures
